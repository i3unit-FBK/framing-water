# Framing Water
This code supports a tangible data-physicalization game that promotes sustainable water use in remote mountain huts. The game functions similarly to a puzzle, asking players to select pieces that relate to the water-consuming activities that they will participate in during an overnight stay. The instructions for building the physical aspects of the Framing Water game can be found **~~TO BE ADDED~~**.


This code was created to work with an Arduino Mega 2560, a 37-pixel addressable RGBW LED strip, and 37 momentary switches actuated by magnets located both on the backs of the game pieces and mounted under the game board. The game runs on 4 AA batteries, and when it is powered on, the 37 LED pixels start in a lit state, are blue in color, and have a brightness of 10, suitable for being played under typical indoor lighting. As game pieces are placed on the game board, a corresponding number of LED pixels turn off to show the remaining quantity of water available for use.
